# Did I closed my door today?: Front-end

## ⚙️ Using
- TypeScript
- Axios
- ReactJS
- Styled-Components
- Polished

## 📝 Instructions
- Install dependencies:
```properties
yarn install
```
Or
```properties
npm install
```

- Run the app with:
```properties
yarn start
```
Or
```properties
npm start
```
